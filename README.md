# Projeto Site WDEV

Projeto desenvolvido utilizando o Framework VueJS. Dados obtidos por meio de APIs utilizando o Axios.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
